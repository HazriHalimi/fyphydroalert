# рџ“– Overview

HydroAlert is a web-based flood monitoring and early warning system designed to assist authorities and the public in monitoring river conditions, rainfall data, and flood risks.

The system provides an interactive dashboard for analysing water level trends, rainfall readings, historical records, and flood alerts to support faster decision-making during flood emergencies.
