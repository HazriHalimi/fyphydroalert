## 📌 Overview

HydroAlert is a web-based flood monitoring and early warning system developed as a Final Year Project (FYP). The system enables monitoring of rainfall data, river water levels, flood risk analysis, and historical records through an interactive dashboard.

---

## ✨ Features

- 🌧 Rainfall Monitoring
- 🌊 River Water Level Monitoring
- 📊 Flood Risk Analysis
- 📍 Station Management
- 🔔 Alert Notification
- 👥 User Authentication
- 📈 Dashboard Analytics
